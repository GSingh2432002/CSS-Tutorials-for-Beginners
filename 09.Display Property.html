Difference between Inline, Block, Inline-block

A)Display:Inline
1.Left & Right margins and padding are allowed, but not on top & bottom.
2.Cannot have a width and height set.
3.other element can sit to their left and right side.
4.a, abbr, Span, strong are some Inline Elements.

B)Display:Block
1.A line break after the block element.
2.Acquire full width if width not defined.
3.article, div, header, hr, main, nav are some Block-level Element.

C)Display:Inline-Block
1.Allow others element to sit to their left and right.
2.Display an element as an inline-level block container.
3.Can Set Height and Width values.
4.It does not add a line break after the element.
